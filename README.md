# Nakhwa Donation Platform

Nakhwa is a full-stack donation management platform that supports blood, food, organ, and financial donations.

## Features
- Blood donation management
- Food donation management
- Organ donation requests
- Financial donations
- User authentication
- Admin dashboard

## Tech Stack
- Backend: Python (Django)
- Frontend: React
- Database: SQLite / PostgreSQL
- API: REST API

## How to Run
1. Clone the repository
2. Install backend requirements
3. Run backend server
4. Run frontend application
